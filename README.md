# Mixin buttons #

> SASS Mixins for generate beautiful buttons

## Installation ##

* **bower**: `bower install mix-buttons`

## Usage ##

```
@import "mix-buttons";

.my-button-class {
  @include button(
    $background-color,
    $font-color,
    $border-radius,
    $font-weight,
    $boolean-button-ghost,
    $boolean-uppercase
  );
}
```
